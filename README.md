# STS2-Custom-Modifiers
A mod for slay the spire 2 that adds a bunch of custom modifiers

## Setup
Clone this repo, and download rider and open this project in it.
Then build it, if your Slay the Spire 2 is in a different directory update the .csproj file accordingly (or just copy the built .dll and .json into your mods folder manually if you're lazy ig).

The only dependency you should need is Net9.0
